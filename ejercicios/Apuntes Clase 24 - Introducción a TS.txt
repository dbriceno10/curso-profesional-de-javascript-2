TypeScript es un super set de JavaScript, tiene múltiples ventajas y restricciones que hacen más seguro que nustro código no va a fallar al momento de ir a producción. Los navegadores no entienes TypeScript, por lo que es necesario usar un transpilador que lo convierta en código JavaScript para que pueda funcionar, en este caso vamos a utilizar parcel.

Luego de instalar parcel con el comando npm install -D parcel-bundler es común que el servidor no levante, una recomendación es borrar las carpetas que se crean .cache y dist, usando el comando rm -rf .cache dist
